# 🎯 智能助手生成您的合成参数 - 快速指南

```bash
python scripts/generate_character.py \

```

## 🚀 使用智能助手的步骤

### 1. 启动智能助手
```bash
python scripts/generate_character.py --ai-assistant
```

### 2. 描述需求（以下任意一种表达方式）
```text
"我要给人物穿上衣服，有两张图片需要合成"
"换装功能，给角色穿上新衣服" 
"图像合成，将服装和人物结合"
"composite模式，合成两张图片"
"穿上这件衣服，生成合成效果"
```

### 3. 回答助手问题
当助手询问风格和质量时，回答：
```text
"写实风格，高质量"          # 对应您的需求
"动漫风格，快速预览"        # 如果想要动漫效果
"像素风格，平衡模式"        # 如果想要游戏风格
```

### 4. 获得智能推荐
智能助手会生成：
```bash
python scripts/generate_character.py \
  --mode-composite \
  --photorealistic \
  --pro \
  --res 4K \
  --auto-complete \
  --composite-instruction "将元素进行智能合成" \
  --preview
```

### 5. 手动补充文件路径
将助手生成的参数与您的具体文件组合：
```bash
python scripts/generate_character.py \
  --mode-composite \
  --composite-images 3.png _extracted_nobg__extracted_clothing_Image_20260124190005_165_113.png \
  --composite-instruction "给这个人穿上这件衣服" \
  --photorealistic \
  --pro \
  --res 4K \
  --auto-complete
```

## ✨ 智能助手的优势

- ✅ **无需记忆参数**：用自然语言描述即可
- ✅ **智能推荐**：根据需求推荐最佳参数组合  
- ✅ **质量优化**：自动配置分辨率和模型参数
- ✅ **多种风格**：支持写实、动漫、像素等各种风格

## 🎯 核心要点

1. **智能助手能识别**：合成模式、风格偏好、质量要求
2. **需要手动补充**：具体的文件路径和合成指令
3. **建议测试**：先用 `--dry-run` 验证参数

这样您就可以轻松生成复杂的图像合成参数了！
