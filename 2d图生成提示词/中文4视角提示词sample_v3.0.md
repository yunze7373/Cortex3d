# 改进的中文四面图生成提示词 (v3.0)

这是基于高成功率提示词的改进版本，融合了"静态物体"和"摄像机环绕"的核心概念。

## 完整提示词

```
生成一份严格的四面视图参考表，共EXACTLY 4个面板。

这是一个几何摄像机环绕任务，不是角色再设计任务。

主体是三维空间中的静态物体。
只有摄像机位置改变。
不允许姿态修正、不允许美学调整、不允许重新解释。

==================================================
## 输出布局（强制要求）
单一图像，恰好4个等大小面板排成一行。

顺序（从左到右）：
[正面 0°] [右侧面 90°] [背面 180°] [左侧面 270°]

图像内不含标签、文字、标记。
==================================================

## 摄像机定义（关键）

摄像机以固定的半径和高度围绕主体环绕。
摄像机目标是主体的原始中心点。

主体不旋转。

--------------------------------------------------
### 四个必需的视图

面板1 — 正面 (0°)：
- 摄像机指向主体的正面
- 主体正面完全可见
- 此视图必须在视觉上与参考图像方向一致

面板2 — 右侧面 (90°)：
- 摄像机位于主体的右侧
- 主体的右侧面对着摄像机
- 主体的正面指向图像左边缘

面板3 — 背面 (180°)：
- 摄像机指向主体的背面
- 主体背面完全可见
- 主体正面完全隐藏

面板4 — 左侧面 (270°)：
- 摄像机位于主体的左侧
- 主体的左侧面对着摄像机
- 主体的正面指向图像右边缘
--------------------------------------------------

==================================================
## 🔒 绝对空间锁定 — 零偏差允许

主体在世界空间中被冻结。

所有空间关系都锁定在身体相对位置，而不是摄像机。

以下内容在所有面板中必须保持100%相同：

- 头部倾斜、头部旋转
- 眼睛方向和凝视角度（不允许"修正"眼神接触）
- 面部表情
- 肩膀角度
- 手臂位置、弯曲角度、手部方向
- 腿部姿态、重心分配、交叉顺序
- 躯干倾斜、扭转、质心
- 衣服褶皱和附着点
- 配饰、武器、道具及其相对位置

❌ 不允许为了可见性调整姿态
❌ 不允许旋转身体面向摄像机
❌ 不允许镜像或交换左右解剖结构
❌ 不允许"修复"各视图中的解剖结构

只允许摄像机旋转导致的透视变化。
==================================================

## 🎨 风格约束
- 电影风格的角色设计
- 与参考图像完全匹配的风格
- 相同的材质、照明氛围和表面细节
- 跨所有面板的一致渲染质量

==================================================
## 背景和环境
- 纯中性灰色或白色背景
- 无缝的工作室风格环境
- 没有可见的地板、地平线、地面纹理或舞台
- 没有转盘、底座、盘子或平台
- 主体应该自然地立在地上，没有可见的几何体

==================================================
## 配置参数
分辨率: 4K
宽高比: 3:2
采样: deterministic（低随机性）

==================================================
## 最终硬性规则

- 恰好4个面板 — 不多不少
- 跨所有面板的相同比例和构图
- 没有重复或镜像视图
- 没有创意解释
- 将主体视为扫描的物理物体

不遵循这些规则是不可接受的。

[角色描述]
```

## 高级变体

### 版本A: 重建导向 (推荐用于3D重建流程)

在原提示词基础上，最后添加：

```
这个任务是为3D重建和建模参考而设计的。
精度比美学更重要。
几何正确性对于下游3D处理至关重要。
```

### 版本B: 摄影风格 (推荐用于photorealistic风格)

修改背景部分为：

```
## 背景和环境
- 纯中性灰色或白色背景
- 专业工作室摄影灯光
- 均匀、漫射的光照（没有刺眼的阴影或戏剧效果）
- 无缝背景
- 主体应该自然地立在地上，没有可见的舞台
```

### 版本C: 严格几何 (推荐用于需要最高精度的场景)

在原提示词最后添加：

```
## 🎯 几何精度要求

这不是角色艺术。这是3D参考文档。

每一毫米都很重要。精度 > 美学。

- 跨面板的几何一致性是唯一重要的指标
- 不允许为视觉吸引力或对称性优化
- 不允许修正感知的解剖问题
- 不允许平滑或理想化形式

将此视为对物理扫描物体的拍摄。
```

## 关键改进点

✅ **认知转换**: 从"角色设计"转向"静态物体扫描"  
✅ **显式摄像机定义**: 固定半径、固定高度、固定中心点  
✅ **极具体的空间锁定**: 逐项列举每个需要保持一致的属性  
✅ **禁止"优化"**: 明确列出模型的常见"好心帮忙"行为  
✅ **配置参数**: 提供分辨率、宽高比、采样策略  
✅ **中性背景**: 消除复杂背景的几何推理困难  
✅ **最终约束**: "不遵循这些规则是不可接受的" 提升权重  

## 使用建议

1. 使用这个提示词与之前的版本对比生成同一个角色
2. 观察四个视角中角色尺寸是否更一致
3. 检查是否仍有"面部朝向摄像机"现象
4. 验证衣服褶皱位置是否在四个视角中稳定
5. 确认背景是否真的是纯中性的
6. 比较生成速度是否有变化
